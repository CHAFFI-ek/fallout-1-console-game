## English Version

**DISCLAIMER: This game is currently available in Russian only. English localization is not yet supported**

This is my text-based RPG project in Python, built on the atmosphere and mechanics of Fallout 1

### What is this project for?
I am writing this game to literally reinforce my knowledge of Python, but that doesn't mean the project won't develop!

### What is already in the game:
* **Fully functional S.P.E.C.I.A.L. just like in fallout!**
* **Global map with triggers**
* **Turn-based combat with action points (AP)**
* **Item and inventory system**

### What I want to add:
* **Dialogues and trading with NPCs**
* **Experience and level system**
* **Fully port the storyline of Fallout 1**
* **And much more!**

## Launch Instructions

### Requirements
* Installed **Python 3.8** or higher

## How to launch:
1. Open the terminal (console) in the root folder of the project
2. Run the command:
```bash
python main.py
```
